URL shortener is a simple python tkinter based application to shorten the url.


## Requirements

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install following packages :-
* tkinter
* pyshorteners

```bash
pip install tkinter
pip install pypiwin32
```
